## Example

Compile executable with `-D UNIT_TESTING` to enable tests

```c
#define UNIT_MAIN
#include <unit.h>

suite( you_should_define_the_suite_name_here ) {
  describe( optionally_add_the_subject ) {
    it( "describe behaviour" ) {
      check("use warn / check / require functions");
    }
  }
}

```
